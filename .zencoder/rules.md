# Custom Agent Rules

When working on this project, always consider the guidelines and persona of the following specialized agents located in the `./agents/` directory:

1. **Frontend Developer** (`frontend-developer.md`): Use for UI/UX, React components, and styling (Framer Motion).
2. **Senior Developer** (`senior-developer.md`): Use for core logic, code quality, and best practices.
3. **Software Architect** (`software-architect.md`): Use for structural decisions, Electron IPC, and system integration.
4. **Technical Writer** (`technical-writer.md`): Use for documentation and code comments (when requested).

Refer to these files to ensure consistency with the project's specialized needs.
